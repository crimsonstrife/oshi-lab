MyOshi Theme Lab

Build and test MyOshi Custom CSS/HTML against a sandboxed profile preview — templates included.

Open the Lab Getting Started All Docs
Runs locally in your browser. No accounts. No uploads. Saved snapshots are local to your browser.

How it works

Read the guide →
  1. 1
    Choose a starting point
    Start from a built-in scaffold OR paste MyOshi preview HTML.
    Template scaffold Real preview + Extract Base
  2. 2
    Iterate in the Lab
    Edit Custom CSS/HTML, checkpoint with snapshots, and verify in the sandboxed preview.
  3. 3
    Export + ship
    Copy/paste into MyOshi when you’re ready.
Quickstart
  1. Open the Lab → choose a template OR Paste preview HTML → click Extract Base.
  2. Edit Custom CSS/Custom HTML and preview.
  3. Use snapshots to checkpoint changes.
  4. Copy/paste into MyOshi or download your bundle.

Tools Palette
In the Lab, open Tools (or press Ctrl/Cmd + K) to generate safer CSS/HTML faster.
  • CSS Scope Fixer (MyOshi auto-scope safe)
  • Accessibility Audit (alt/labels contrast checks)
  • Theme Variables (quick palette presets)
  • Component Styler (cards/nav/tabs/link styling)
  • Schedule Builder (stream schedule markup)
  • Color Converter (HEX/RGBA snippets)
