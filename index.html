MyOshi Theme Lab

Build and test MyOshi Custom CSS/HTML against a sandboxed profile preview — templates included.

Open the Lab Getting Started All Docs
Runs locally in your browser. No accounts. No uploads. Saved snapshots are local to your browser.

How it works

Read the full guide →
  1. 1
    Choose a starting point
    Start from a built-in scaffold or paste MyOshi preview HTML.
    Template scaffold Real preview + Extract Base
  2. 2
    Iterate in the Lab
    Edit Custom CSS/HTML, checkpoint with snapshots, and verify in the sandboxed preview.
  3. 3
    Export + ship
    Copy/paste into MyOshi or download a bundle when you’re ready.
Quickstart
  1. Open the Lab → choose a template.
  2. Paste preview HTML → click Extract Base.
  3. Edit Custom CSS/Custom HTML and preview.
  4. Use snapshots to checkpoint changes.
  5. Copy/paste into MyOshi or download your bundle.
