## Core idea

MyOshi is currently private, so the lab **does not fetch live profiles**.

Instead, it ships with curated **demo templates** that resemble the real layout closely enough to:
- develop themes quickly,
- avoid breaking changes in app code,
- keep templates readable and maintainable.

---

## Recommended workflow

1. **Load Demo Template** in the Theme Lab
2. Write **Custom CSS** to style the page
3. Add **Custom HTML** when you need extra structure (widgets, wrappers, layouts)
4. Save **Snapshots** as you iterate
5. Copy **Custom CSS** + **Custom HTML** into MyOshi

---

## Safety model

The preview runs inside a sandboxed iframe:
- scripts do not execute
- this reduces risk while experimenting with third-party snippets

> You are still responsible for what you paste into MyOshi.
> Prefer CSS-only solutions whenever possible.
