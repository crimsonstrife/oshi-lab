Widgets are small pre-built blocks of **HTML + CSS** you can drop into your theme via **Tools → Widget Inserter**.

## How widgets are designed

- Most widgets use `labw-*` class names so their CSS doesn’t accidentally style other MyOshi elements.
- A few widgets are marked “native” and intentionally reuse MyOshi/base-template classes (like `.card`).

## Recommended workflow

1. Use **Widget Inserter** to insert the widget into **Custom HTML** and **Custom CSS**.
2. Customize the widget fields in the tool (or edit the inserted HTML/CSS directly).
3. Keep widget CSS together so you can remove/replace it later.

## Widget catalog

- Callout Box
- Info Card
- MyOshi Card (native-looking)
- Notice Glow (native)
- Sparkle Divider (native)
- Streaming Schedule (Grid)
- Twitch Buttons
