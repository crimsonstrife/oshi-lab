About Oshi-Lab

Oshi Theme Lab is a browser-based workspace for building, testing, and refining custom MyOshi CSS and HTML in a safer sandboxed preview.

Quick Links
Open the Lab Read the Docs Join the Community View on GitHub
What is this place?

Oshi-Lab exists to make MyOshi customization easier to experiment with. Instead of editing blindly, you can test layouts, styles, widgets, and helper tools in one place before copying them into your profile. It also provides some starter templates and tools for users new to CSS and HTML.

The goal is to make theme building more approachable for newer users while also giving more experienced creators faster workflows, reusable snippets, and better tooling.

Note: Oshi-Lab is an independent project and is not affiliated with MyOshi or its parent company. I do this project as a hobby, if you'd like to support me, please check out my socials and other projects!
